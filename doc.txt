Project: Smart Complaint Management System
Description

This is a full-stack web application built using React (frontend), Node.js/Express (backend), and MongoDB (database) with an NLP module (Python/Flask API or Node.js library) for sentiment analysis.

The system allows students/residents/employees to lodge complaints (e.g., hostel, college, office issues).
The AI module automatically analyzes the complaint text and assigns an urgency priority (High, Medium, Low) using sentiment detection and keywords.

Admins get a dashboard to track complaints, filter by status/priority, and view analytics on trends and resolution performance.
